PetStore.DatabaseController

Описание: Контроллер для управления базой данных через консольный интерфейс

Обеспечивает взаимодействие между пользователем и DatabaseService. Все изменения автоматически сохраняются в файл базы данных.

Методы

PetStore.DatabaseController.#ctor(System.String)
Инициализирует новый экземпляр DatabaseController
_filePath: Путь к файлу базы данных (.xlsx)
Исключения:
  • T:System.IO.FileNotFoundException - Если файл не существует
PetStore.DatabaseController.viewDatabase
Отображает всю базу данных в консоли
Примечания:

Формат вывода: - Таблица животных (ID, Вид, Порода) - Таблица клиентов (ID, Имя, Возраст, Адрес) - Таблица продаж (ID, ID Животного, ID Клиента, Дата, Цена)

PetStore.DatabaseController.addAnimal
Добавляет новое животное через консольный интерфейс
Примечания:

Последовательно запрашивает: 1. ID животного (целое число) 2. Вид животного (строка) 3. Породу животного (строка)

PetStore.DatabaseController.updateAnimal
Обновляет данные животного через консольный интерфейс
Примечания:

Запрашивает ID животного, затем новые значения вида и породы

PetStore.DatabaseController.deleteAnimal
Удаляет животное через консольный интерфейс
Примечания:

Запрашивает ID животного для удаления

PetStore.DatabaseController.addCustomer
Добавляет нового клиента через консольный интерфейс
Примечания:

Последовательно запрашивает: 1. ID клиента (целое число) 2. Имя (строка) 3. Возраст (целое число) 4. Адрес (строка)

PetStore.DatabaseController.updateCustomer
Обновляет данные клиента через консольный интерфейс
Примечания:

Запрашивает ID клиента, затем новые значения имени, возраста и адреса

PetStore.DatabaseController.deleteCustomer
Удаляет клиента через консольный интерфейс
Примечания:

Запрашивает ID клиента для удаления

PetStore.DatabaseController.addSale
Добавляет новую продажу через консольный интерфейс
Примечания:

Последовательно запрашивает: 1. ID продажи (целое число) 2. ID животного (целое число) 3. ID клиента (целое число) 4. Дата продажи (в формате ГГГГ-ММ-ДД) 5. Цену (десятичное число)

PetStore.DatabaseController.updateSale
Обновляет данные о продаже через консольный интерфейс
Примечания:

Запрашивает ID продажи, затем новые значения всех полей

PetStore.DatabaseController.deleteSale
Удаляет продажу через консольный интерфейс
Примечания:

Запрашивает ID продажи для удаления

PetStore.DatabaseController.executeQueries(System.Int32)
Выполняет аналитические запросы к базе данных
_i: Номер запроса (1-4)
Примечания:

Доступные запросы: 1 - Список всех собак 2 - Количество продаж по имени клиента 3 - Виды животных, купленных клиентами старше 50 лет 4 - Общая стоимость попугаев Ара, купленных жителями Уфы

PetStore.DatabaseController.logAction(System.String)
[Приватный] Записывает действие в лог-файл
message: Сообщение для логирования
Примечания:

Лог сохраняется в файл log.txt в формате: [Дата Время]: Сообщение
