PetStore.DatabaseService

Описание: Основной сервис для работы с базой данных магазина животных

Использует Excel-файл в качестве хранилища данных. Содержит три листа: Animals (ID, Species, Breed), Customers (ID, Name, Age, Address), Sales (ID, AnimalID, CustomerID, Date, Price).

Методы

PetStore.DatabaseService.#ctor(System.String)
Инициализирует новый экземпляр DatabaseService
_filePath: Путь к Excel-файлу с базой данных
Исключения:
  • T:System.IO.FileNotFoundException - Выбрасывается если файл не существует
PetStore.DatabaseService.getAnimals
